# Firebase Setup Instructions

1. Go to https://console.firebase.google.com/
2. Create a new project (e.g., Fantasy T11 Cricket)
3. Go to Project Settings > Add Web App
4. Copy the Firebase config and paste it in js/firebase.js (replace YOUR_API_KEY etc.)
5. Go to Realtime Database and click "Create Database"
6. Set security rules to test mode or authenticated access as needed
7. Upload folder to your web host or use Firebase Hosting
